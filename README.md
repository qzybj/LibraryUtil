# libutil
整合的工具类

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	    compile 'com.github.qzybj:libutil:0.2'
	}
